# Custom GIFs

## Instructions

If you want to use your own GIFs instead of Giphy links, place them here.

## Current GIFs Used (from Giphy)

The website currently uses these Giphy GIFs:
1. Cute hearts animation (header)
2. Love animation (main section)
3. Celebration animation (success page)
4. Couple animation (success page)

## Adding Your Own GIFs

1. Place GIF files in this folder
2. Update the `src` attributes in `index.html`:

```html
<!-- Example -->
<img src="assets/gifs/your-animation.gif" alt="Description">
```

## GIF Optimization

To keep loading times fast:
- **Size**: Keep each GIF under 1-2 MB
- **Dimensions**: Resize to appropriate size (400-600px width)
- **Optimization Tool**: https://ezgif.com/optimize

## Finding Pinterest-Style GIFs

- **Giphy**: https://giphy.com/ (search "cute couple", "love", "hearts")
- **Tenor**: https://tenor.com/
- **Pinterest**: https://pinterest.com/ (download and convert to GIF)

**Tip**: Pastel-colored and minimalist GIFs match the website aesthetic best!
